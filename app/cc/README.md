### TensorFlow & TensorFlow-Lite C/C++ API Benchmark App
#### Run Benchmark
```bash
# TF C API x86_64 cpu
./run.sh x86_64 cpu
# TF C API  x86_64 gpu
./run.sh x86_64 gpu
# TFLite C++ aarch64
./run.sh aarch64|armv7l
# TFLite C++ armv7l
./run.sh armv7l
```
